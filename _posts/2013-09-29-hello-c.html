Run Dev C++ from start menu.

Press ctrl+n.
[ctrl+n is standard command in for computer programs, which creates new instance, in our case new file]

Type this.

#include<iostream>

using namespace std;

int main()
{
   cout<<"Hello C++";
 
   int hold;
   cin>>hold;
 
   return 0;
}

Execute->Compile & Run. [Or you can press f11 (better way)]

Bingo!!
